A Pythagorean triplet is a set of three natural numbers, a < b < c, for which,

a2 + b2 = c2

For example, 32 + 42 = 9 + 16 = 25 = 52.

There exists exactly one Pythagorean triplet for which a + b + c = 1000.
Find the product abc.

Key

Euclid's Formula allows you to generate a Pythagorean triple using two positive integers m and n, where m > n.

Using Euclid's formula:

  • a = m2 - n2

  • b = 2mn

  • c = m2 + n2
By iterating through values of m and n, where each value is less than the sum S,
and m > n, one can find a pythagorean triple where a + b + c = S.
